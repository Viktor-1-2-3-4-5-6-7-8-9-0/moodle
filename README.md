# Калькулятор квадратных уравнений для Moodle

Плагин-блок для Moodle, позволяющий пользователям решать квадратные уравнения вида `ax² + bx + c = 0` прямо на сайте. Поддерживает сохранение истории вычислений.


# Функционал

- Ввод коэффициентов `a`, `b`, `c`
- Автоматическое решение уравнения с выводом:
  - Два действительных корня
  - Один действительный корень
  - Нет действительных корней
- История вычислений (последние 10 записей)
- Поддержка нескольких языков: **English**, **Русский**


# Структура папок
blocks/calculator/
├── classes/ → основные классы (QuadraticSolver, CalculatorLog)
├── form/ → форма ввода коэффициентов
├── output/ → рендеринг UI и истории
├── db/ → установка таблицы логов
├── lang/ → языковые строки (en, ru)
├── history.php → страница истории
├── block_calculator.php → основной блок
├── styles.css → стили
├── version.php → версия плагина
└── README.md ← этот файл


# Установка

1. Скачай архив `install.zip` или клонируй репозиторий.
2. Распакуй содержимое в папку `blocks/calculator/` в корне Moodle.
3. Перейди в админку Moodle → **Управление сайтом → Плагины → Блоки → Установка плагинов**.
4. Нажми «Обновить» или перезагрузи страницу — плагин должен появиться.
5. Добавь блок на нужную страницу (например, на главную).


## История вычислений

Все вычисления сохраняются в базе данных и доступны по ссылке **«View calculation history»** в блоке. История показывает последние 10 записей.


# Языки

Поддерживаемые языки:
- Английский (`en`)
- Русский (`ru`)

Чтобы добавить новый язык — создай папку `lang/xx/` и файл `block_calculator.php` с переводами.
